Функциональные требования
1. Каталог меню
Пользователь должен иметь возможность просматривать список блюд, сортировать их по категориям и добавлять в корзину.
Каждая позиция меню должна содержать: название, описание, цену, изображение, кнопку "Добавить в корзину".Пример:
Название: "Шаурма бургер"
Описание: "Шарума бургер. просто бургер"
Цена: "350 рублей"
2. Форма заказа
Пользователь заполняет данные о заказе (адрес доставки, контактный телефон) и подтверждает заказ.
Выбор способа доставки: доставка или самовывоз.
Корзина позволяет изменять количество товаров, удалять товары, применять скидки/купоны.
3. Онлайн-оплата
Пользователь выбирает способ оплаты, вводит данные карты, и система проверяет корректность транзакции.
Возможность использования нескольких платежных систем (например, Visa, MasterCard, МИР).
Обеспечение безопасности платежей (SSL/TLS шифрование).
4. Личный кабинет пользователя
Пользователь должен иметь возможность просматривать историю заказов, повторно оформлять заказ, изменять свои данные.
Сценарий использования:
Пользователь входит в личный кабинет.
Открывает вкладку "Мои заказы".
Выбирает заказ для повторного оформления.
5. Контактная информация и форма обратной связи
Контакты пиццерии включают адрес, номер телефона, электронную почту.
Форма обратной связи позволяет пользователям отправлять запросы и предложения.
Нефункциональные требования
Скорость загрузки страниц: страница меню должна загружаться за 2 секунды.
Доступность: сайт должен корректно отображаться на экранах с разрешением от 320x480 до 1920x1080 пикселей.
Безопасность: использование HTTPS для защиты данных пользователей.
Совместимость: поддержка современных браузеров (Google Chrome, Mozilla Firefox, Safari).
Ограничения
Поддержка двух языков: русский и английский.
Работа только в современных браузерах (Google Chrome, Mozilla Firefox, Safari).
Отсутствие поддержки старых версий операционных систем (Windows XP, iOS ниже версии 10).
Алгоритмы
Алгоритм оформления заказа
Пользователь добавляет товар в корзину.
Выбирает адрес доставки или пункт самовывоза.
Указывает способ оплаты.
Система проверяет корректность данных и отправляет заказ.
