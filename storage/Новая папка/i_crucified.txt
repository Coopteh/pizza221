1. Клиент
Атрибут	Тип данных	Описание
ID_клиента	Integer	Уникальный идентификатор клиента
Имя_клиента	String	Имя клиента
Телефон	String	Номер телефона клиента
Адрес	String	Адрес доставки
Email	String	Электронная почта клиента
Пароль	String	Хэшированный пароль
2. Заказ
Атрибут	        Тип данных	Описание
ID_заказа	Integer	        Уникальный идентификатор заказа
Дата_оформления	DateTime	Дата и время оформления заказа
Статус	        Enum	        Статус заказа (например, "новый", "в пути")
Сумма	        Decimal	        Общая сумма заказа
ID_клиента	Foreign Key	Идентификатор клиента, сделавшего заказ
3. Шаурма
Атрибут	       Тип данных	Описание
ID_шаурмы      Integer	        Уникальный идентификатор пиццы
Название       String	        Название пиццы
Описание       Text	        Краткое описание пиццы
Цена	       Decimal	        Цена одной порции пиццы
Калорийность   Integer	        Калорийность одной порции пиццы
Вес	       Decimal	        Вес одной порции пиццы
Изображение    Blob/URL	        Изображение пиццы
Время_приготовления	Time	Время приготовления пиццы
4. Состав ингредиентов
Атрибут	Тип данных	Описание
ID_ингредиента	        Integer	Уникальный идентификатор ингредиента
Название_ингредиента	String	Название ингредиента
Количество	        Decimal	Количество ингредиента в пицце
Единица_измерения	String	Единица измерения количества
ID_шаурмы	        Foreign Key Идентификатор пиццы, к которой относится ингредиент
